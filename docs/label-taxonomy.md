# Label Taxonomy

RepoCompass uses GitHub labels to categorize issues and pull requests.

## Triage Labels
- `needs-triage`: Newly opened issues that require maintainer review.
- `needs-information`: Issue requires more details from the author.

## Type Labels
- `bug`: Something isn't working.
- `enhancement`: New feature or request.
- `analyzer`: Request for a new codebase analyzer.
- `documentation`: Improvements or additions to documentation.

## Status Labels
- `good first issue`: Good for newcomers.
- `help wanted`: Extra attention is needed.
- `wontfix`: This will not be worked on.
