# Good First Issues Backlog

This is a seed backlog of ideas that can be converted into GitHub issues with the `good first issue` label.

1. **Add Python Requirements Analyzer**: Check if `requirements.txt` exists but `pyproject.toml` is missing.
2. **Add License Analyzer**: Check if `LICENSE` file exists at the root.
3. **Add Dependabot Analyzer**: Check if `.github/dependabot.yml` exists.
4. **Markdown Renderer**: Add a table of contents to the Markdown renderer output.
5. **Fixtures**: Add a `basic-rust-repo` fixture containing a `Cargo.toml`.
