# Support

Welcome to the RepoCompass community! If you need help, here are the best places to look:

## 1. Documentation
Our [docs/](docs/README.md) directory contains architecture walkthroughs, contributor guides, and extension documentation.

## 2. GitHub Discussions
For general questions, "How do I do X?" or proposing a new architecture change, please open a [GitHub Discussion](https://github.com/JustCallMeMin/repoCompass/discussions).

## 3. GitHub Issues
If you have found a bug, please use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template to file an issue.

## 4. Paid Support
At this time, RepoCompass is an entirely community-driven OSS project. We do not offer SLAs or paid support.
