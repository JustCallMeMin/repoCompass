# Changelog

All notable changes to RepoCompass will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core scanning engine with support for basic Git and local directory resolution.
- Snapshot system to isolate file reading.
- Support for `readme`, `ci`, `contributing`, and `scripts` analyzers.
- Markdown and JSON report renderers.
- Postgres-backed history storage.
- Extensible API for running and querying scans.
- Next.js dashboard for visualizing scan history.
- Milestone 7: Open Source Maturity workflows (Docs, CI, Testing, Community).
