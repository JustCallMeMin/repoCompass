# Security Policy

## Supported Versions

Currently, RepoCompass is in pre-release (`v0.x`). Only the `main` branch and the latest minor release are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.y   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within RepoCompass, please do **NOT** open a public issue.

Instead, please send an email to [security@repocompass.io]. All security vulnerabilities will be promptly addressed. We aim to acknowledge receipt of your vulnerability report within 48 hours and to provide regular updates on our progress in fixing the issue.
